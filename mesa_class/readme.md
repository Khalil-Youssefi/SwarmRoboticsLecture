# Mesa Code

Two Mesa models are implemented here, using Mesa 3.5.1 and Python 3.14. To get started, install the packages listed in `requirements.txt` into a virtual environment.

## Discrete

The `discrete` folder contains a Mesa model of randomly moving agents in a discrete grid environment. Their goal is to visit each cell at least once.

## Continuous

The `continues` folder contains a Mesa model of PSO agents organized into two swarms. Their goal is to find the minimum of the Rastrigin function.
